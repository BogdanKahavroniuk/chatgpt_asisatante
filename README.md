## *To run project*

 **Create virtual environment :**

    python -m venv venv
           source venv/bin/activate      # On macOS/Linux
           venv\Scripts\activate\# Windows

 **Install dependencies :**

    pip install -r requirements.txt

**Add your OpenAI API key :**

    Create a .env  file:

    OPENAI_API_KEY=your_api_key_here
